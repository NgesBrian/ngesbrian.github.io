VIPER

A Private Information Retrieval systems based on Homomorphic Encryption. It supports efficient retrieval of multi-attribute database records together with a lightweight integrity verification mechanism.

Website

Accelerated MiMC Hardware Architecture for Zero-Knowledge Proof systems on the Edge. Finalist for Top Picks in Hardware and Embedded Security 2025.

GitHub

Optimized PQC primitives (ML-KEM and ML-DSA) for high-performance execution across NIST FIPS security levels. Developed co-design architectures achieving up to 60% performance improvements.
